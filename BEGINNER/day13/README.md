# Day 13 — Collections II: Sets, Sorting, and Streams

## Objectives
- Use `HashSet` and `TreeSet` for unique-element collections
- Sort with `Comparable` (natural ordering) and `Comparator` (custom ordering)
- Use Java Streams for data processing: `.stream().filter().map().collect()`

## Concepts
- **Set** — no duplicates; `HashSet` (fast, unordered), `TreeSet` (sorted)
- **Comparable** — implement `compareTo()` in the class itself (natural order)
- **Comparator** — external comparison logic (lambda or class)
- **Stream pipeline** — `source → intermediate ops → terminal op`
