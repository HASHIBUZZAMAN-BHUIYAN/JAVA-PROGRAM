# Day 10 — OOP III: Abstract Classes and Interfaces

## Objectives
- Define and use abstract classes
- Define and implement interfaces
- Understand the difference: abstract class vs interface

## Key Difference

| | Abstract Class | Interface |
|---|---|---|
| Fields | Any type | `public static final` only |
| Methods | Can have concrete + abstract | Default abstract (or `default`/`static`) |
| Constructor | Yes | No |
| Extends | One only | Can implement many |
| Use when | Sharing code between related types | Defining a contract/capability |
