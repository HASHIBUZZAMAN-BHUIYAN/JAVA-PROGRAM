# Day 14 — Capstone: Library Management System

## What You Build
A command-line Library Management System that combines everything from Days 1-13:
- **Classes + OOP** (Book, Member classes)
- **Inheritance** (PhysicalBook, EBook extend Book)
- **Collections** (ArrayList for books and members, HashMap for checkouts)
- **Exception handling** (custom exceptions for overdue/not-found cases)
- **File I/O** (save/load library data to a text file)
- **Streams** (search, filter, report generation)

## Files
- `Lesson.java` — feature walkthrough demo (runs automatically)
- `MiniProject.java` — full interactive CLI application
- `Exercises.java` — practice adding new features
- `Solutions.java` — exercise solutions

## How to Run
```
javac *.java
java Lesson        # walkthrough demo
java MiniProject   # interactive library system
```

## Data File
`library_data.txt` is created in the current directory when you save from MiniProject.

## Time Estimate
90–120 minutes
