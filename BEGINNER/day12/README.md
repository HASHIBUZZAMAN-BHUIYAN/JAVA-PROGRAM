# Day 12 — Collections I: ArrayList and HashMap

## Objectives
- Use `ArrayList<T>` — a resizable list
- Use `HashMap<K,V>` — a key-value store
- Iterate over collections with for-each and iterators
- Understand why collections beat arrays for most real-world use

## Key Types
- `ArrayList<String>` — ordered, allows duplicates, dynamic size
- `HashMap<String, Integer>` — unordered, unique keys, fast lookup
- `List`, `Map` — the interface types (always declare with the interface)

## How to Compile and Run
```
javac *.java
java Lesson
```
