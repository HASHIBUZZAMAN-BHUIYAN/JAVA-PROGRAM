# Day 09 — OOP II: Inheritance and Polymorphism

## Objectives
- Use `extends` to inherit from a parent class
- Call parent constructors and methods with `super`
- Override methods with `@Override`
- Understand polymorphism: a parent reference can hold a child object

## Concepts
- **Inheritance** — child class gets all fields/methods of parent
- **`super()`** — calls parent constructor (must be first line)
- **`@Override`** — tells the compiler you intend to override a parent method
- **Polymorphism** — `Animal a = new Dog()` — works because Dog IS-AN Animal
- **`instanceof`** — check an object's actual type at runtime
