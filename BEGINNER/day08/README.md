# Day 08 — OOP I: Classes and Objects

## Objectives
- Define a class with fields, constructors, and methods
- Create objects with `new`
- Use `this` to refer to the current object
- Implement encapsulation: private fields + public getters/setters

## Concepts
- **Class** — a blueprint/template
- **Object** — an instance of a class
- **Constructor** — special method called when object is created
- **Encapsulation** — hide internal data, expose only what's needed
- **this** — refers to the current object inside a method/constructor

## How to Compile and Run
```
javac *.java
java Lesson
```
