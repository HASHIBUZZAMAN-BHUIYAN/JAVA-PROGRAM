# Day 06 — Arrays

## Objectives
- Declare and initialize 1D and 2D arrays
- Access elements by index, iterate with loops
- Use `array.length`, `Arrays.sort()`, `Arrays.toString()`
- Understand arrays are fixed-size in Java

## Concepts
- `int[] arr = new int[5]` — creates array of 5 zeros
- `int[] arr = {1, 2, 3}` — array literal
- `arr[0]` — zero-based indexing
- `arr.length` — number of elements
- 2D array: `int[][] matrix = new int[rows][cols]`

javac *.java
