# Day 05 — Methods

## Objectives
- Declare and call methods with parameters and return values
- Understand method overloading (same name, different parameters)
- Understand variable scope (local vs field)
- Use `static` methods in a utility style

## Concepts
- `public static returnType methodName(params)` — basic method signature
- `void` — method returns nothing
- `return value` — sending a value back to the caller
- **Overloading** — multiple methods with the same name but different parameter lists
- **Scope** — local variables exist only inside their method/block

## How to Compile and Run

```
javac *.java
java Lesson
```
Or double-click `build.bat`.
